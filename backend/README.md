# backend_api_for_u,_a_react-based_frontend_application._the_api_will_provide_data_storage,_retrieval,_and_manipulation_capabilities_for_the_frontend_application. Backend API

Complete FastAPI backend generated from report analysis.

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Set up environment:
```bash
cp .env.example .env
```

3. Run the application:
```bash
python main.py
```

## API Documentation

Visit http://localhost:8000/docs for Swagger UI documentation.
